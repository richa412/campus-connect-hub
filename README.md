# Campus Connect

Campus Connect is a student-focused social platform designed to help college students connect, explore opportunities, and engage with campus activities in one place.

This project is currently focused on frontend development and authentication workflows while gradually evolving into a larger full-stack application.

---

## Features

* Student authentication
* Login and signup pages
* Campus feed UI
* Events page
* Marketplace page
* Opportunities section
* Messaging interface
* Student profile page
* Responsive modern UI

---

## Tech Stack

* React
* TypeScript
* Tailwind CSS
* React Router
* Supabase Authentication
* shadcn/ui
